Python project 3

This code is designed to find and print the top three largest numbers from a set of ten user inputs. It defines three global variables (Bigger1, Bigger2, Bigger3) to store these numbers. The big() function updates these variables by comparing each new input with the existing values. The program prompts the user to enter a number ten times, processing each input through the big() function. If a non-numeric value is entered, the program prints an error message and stops. Finally, the program prints out the three largest numbers in descending order.

Python project 4

The code is designed to manage customer billing for SMS and call services. It includes functions to calculate the cost of local and international SMS and calls, add and store customer data, retrieve a customer's final bill based on their phone number, identify the highest bill and the associated phone number, find the name with the highest bill, list customers sorted by their bill amount in descending order, and retrieve the names of customers with the top three highest bills. It also allows users to update customer information based on their name. The main() function provides a menu for users to interact with these features.
